# Repositorio

Un repositorio personal de links con estética cyberpunk/retro. Hecho para uso propio, pero de código abierto.

## Características

- Links organizados por categorías
- Búsqueda y filtros
- Reproductor de música
- Notas rápidas (comandos, fragmentos)
- Vista grid, lista y compacta
- Fijado de links favoritos
- Login de administrador vía Firebase

## Stack

HTML + CSS + JS vanilla. Firebase Realtime Database para persistencia.

## Uso

Abre `index.html` con un servidor local o despliega en cualquier hosting estático.

Para administrar los links, necesitás configurar tu propio proyecto de Firebase y actualizar las credenciales en `app.js`.
